# Bug Hunt - Debug Lab

Repositório da disciplina **Testes e Qualidade de Software - UNICAP**.

## Objetivo

Praticar técnicas de:

- Debug
- Identificação de defeitos
- Reporte de bugs

## Atividade

1. Execute os scripts da pasta `src`
2. Identifique os bugs
3. Utilize técnicas de debug
4. Registre o defeito utilizando o template de bug report

## Técnicas sugeridas

- Print Debugging
- Logging
- Breakpoints
- Divisão e conquista
